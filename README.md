# 🌊 Progetto Beachwear Sostenibile (TypeScript)

Questo progetto simula un sistema di gestione per un brand di abbigliamento mare ecosostenibile. Sviluppato in **TypeScript**, il sistema modella il ciclo di vita di un prodotto, dalla produzione all'assegnazione al cliente finale, applicando i principi della programmazione orientata agli oggetti (OOP).

## 🚀 Obiettivi del Progetto
Dimostrare la capacità di strutturare un'architettura software solida utilizzando:
- **Interfacce**: Per definire i contratti dei dati.
- **Classi**: Per gestire la logica di business e lo stato degli oggetti.
- **Tipizzazione Forte**: Per prevenire errori a runtime e garantire coerenza.

## 📁 Struttura della Repository
In linea con le best practice di sviluppo, la repository contiene solo il codice sorgente:

- `src/`: Contiene i file sorgente TypeScript (`.ts`).
  - `Interfaces.ts`: Definizione dei contratti (IProdotto, ICliente, ecc.).
  - `Classes.ts`: Implementazione delle logica e delle classi.
  - `index.ts`: Script di test e inizializzazione.
- `tsconfig.json`: Configurazione per la compilazione TypeScript.
- `.gitignore`: Configurato per escludere `node_modules` e la cartella di build `dist`.
